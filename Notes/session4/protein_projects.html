Useful information

Calculating the number of organic molecules to reach a certain concentration

To calculate the volume of the simulation box that is occupied by the solvent. The height of the solvent layer can be obtained by subtracting the height of the clay unit cell from the total height of the expanded simulation box.
Remember to keep the units of the volume and the concentration to be consistent in your calculations.

Calculating the linear density of organic molecules

Use gmx density to calculate the linear densities of molecules or ions along the z-axis.
Usage:

gmx density ‐f <trajectory> ‐s <tpr file> ‐n <index file> ‐o <output file> ‐d z ‐sl <number of slices> ‐ng 2 ‐dens number ‐b <beginning time> ‐e <end time>>

‐f specifies the input trajectory file (.xtc or .trr)

‐s specifies the input .tpr file

‐n specifies the input index file (.ndx)

‐o specifies the output file (.xvg)

‐d specifies the direction of the density calculation

‐sl specifies the number of slices along the density profile. Setting it to 1‐2 times the box height in Å usually gives a good resolution.

‐ng specifies the number of groups to calculate the density for

‐dens specifies the type of density to calculate (number, mass, charge, etc.)

‐b specifies the beginning time of the trajectory to consider

‐e specifies the end time of the trajectory to consider

Before running gmx density you will need to use gmx make_ndx to create an index file with the atoms of interest.

Input files

Organic molecule structures and molecule definitions are located inside the STRUCTURES folders of each project. Force fields are inside the FF and clay unit cells inside the CLAYS folders. Run parameter .mdp files are inside MDP. Use em.mdp for energy minimisation, nvt.mdp for equilibration at constant temperature and volume, npt.mdp for equilibration at constant temperature and pressure and prod.mdp for the production run. Make sure to fill in any missing values for the keywords of the .mdp files that are marked with COMPLETE HERE.

You will find template scripts for submitting your simulation on eddie inside the project folder.
Remember to run all of your simulations on eddie!
